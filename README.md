## 🚀 Feature Spotlight — Automation-Generated Hire Button

The centerpiece of this site is a **live demonstration of AI-driven UI generation**:

```
User visits desidevloper.com
        │
        ▼
Automation workflow triggers
        │
        ▼
Claude API evaluates availability context
        │
        ▼
Dynamic "Hire Me" CTA is rendered — content, style & state generated at runtime
        │
        ▼
Button click → WhatsApp / Email / Calendar booking flow
```

**Why this matters:**
- 🤖 The button **text, color state, and availability label** are automation-controlled
- 🔄 Can toggle between `Open for Work` / `Limited Availability` / `Fully Booked` without touching code
- 📡 Powered by the same LLM + workflow stack I build for clients

## 🤖 How the Automation Works

The hire button is controlled by a lightweight serverless function + Claude API call:

```typescript
// api/availability.ts (Vercel Edge Function)
export default async function handler(req, res) {
  const response = await fetch("https://api.anthropic.com/v1/messages", {
    method: "POST",
    headers: {
      "x-api-key": process.env.ANTHROPIC_API_KEY,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      model: "claude-sonnet-4-20250514",
      max_tokens: 100,
      messages: [{
        role: "user",
        content: `Given current workload context: ${JSON.stringify(workloadState)}, 
                  return JSON with: { status: 'open'|'limited'|'booked', label: string, cta: string }`
      }]
    })
  });

  const data = await response.json();
  return res.json(JSON.parse(data.content[0].text));
}
```
